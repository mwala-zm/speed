## Getting Started
Let's get started...
cd to the cloned repo and follow the steps below:

1. ```shell
   git clone https://github.com/mwala-zm/speed.git
   ```
   Open virtual environment.
2. ```shell
    pipenv shell
    ```
   Install dependencies.
3. ```shell
    pipenv install
    ```
   To run the app.
4. ```shell 
    python main.py
    ```
Have fun
